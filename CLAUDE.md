# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Environment

- Python 3.13 (CPython), managed via `.venv` at the project root
- Activate: `source .venv/bin/activate`
- Run a script: `python main.py`
- Install a dependency: `pip install <package>`

## Project state

This project was created from a PyCharm default template and has no application code yet. `main.py` is the entry point placeholder.

## Database

The database schema is defined in `prisma/schema.prisma`. Reference it whenever you need to understand data structure.

## Code style

Use comments sparingly. Only comment complex code.
